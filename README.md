 دليل أوامر استخدام AlZill

هنا شرح لجميع الخيارات المتاحة في الأداة:

### 1. عرض المساعدة
`./azil`
يعرض شاشة المساعدة (usage) للتعرف على الخيارات المتاحة.

### 2. الفحص السريع
`./azil -u "https://example.com"`
بدء فحص ويب سريع باستخدام الإعدادات الافتراضية.

### 3. الفحص المفصل (Verbose)
`./azil -u "https://example.com" -v`
يطبع جميع خطوات الزحف وردود الخادم بالتفصيل.

### 4. وضع الهجوم (Attack Mode)
* `azil -u "URL" --attack`: فحص مع هجوم Logic Abuse.
* `azil -u "URL" --deep --attack`: فحص عميق وشامل مع تفعيل الهجوم.

### 5. التحكم في الفحص
* `--no-infra`: تعطيل الفحص التلقائي للبنية التحتية.
* `-d 2.5`: تحديد تأخير مخصص (Delay) بين الطلبات.

### 6. اختبار حزم الشبكة (Network Test)
> **تحذير:** لا تستخدم هذه الأوامر إلا في بيئة اختبارية.
* **UDP:** `./azil --ip 127.0.0.1 --port 53 --count 10 --mode udp`
* **TCP:** `./azil --ip 127.0.0.1 --port 80 --count 100 --mode tcp`
* **Wide:** `./azil --ip 127.0.0.1 --port 53 --count 50 --mode wide` (لإثقال النظام المستهدف).

### 7. استخدام حمولات (Payloads) خارجية
`./azil -u "URL" --messages-url "https://site.com/payloads.txt"`
لجلب قوائم اختبار خارجية أو ملفات محلية.

### 8. إغراق المنافذ المفتوحة
`./azil -u "URL" --port 8080 --count 100 --mode udp -v`
يستخدم لإغراق منفذ معين تم اكتشافه حتى يتم إغلاقه.
